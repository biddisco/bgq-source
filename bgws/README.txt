
This top level directory is the Blue Gene Web Services,
which provides a RESTful web service for Blue Gene.


Contact: Brant Knudson <bknudson@us.ibm.com>


bgws is used by the Blue Gene Navigator (but does not depend on it to build).

bgws requires
o Scheduler APIs in hlcs
o DB APIs in db
o Log in utility
o Log4cxx in extlib
